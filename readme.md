## Tech Stack
- MongoDB, Express.js, React.js, Node.js
- JWT Authentication
- Tailwind CSS

## Features
- Role-based access (Renter, Owner, Admin)
- Property listing with search and filters
- Booking request and approval workflow
- Admin approval system for owners
- Full CRUD operations on properties

## Roles
**Renter** — Browse properties, send booking requests, track booking status
**Owner** — Add/edit/delete properties, manage booking requests (requires admin approval)
**Admin** — Approve/reject owners, oversee all users, properties, and bookings
